Create "Venice Tips and Suggestions" Menu Item

Open your browser console on your running app and paste this code:

// Create the menu item
fetch('/api/admin/menu', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    title: { en: 'Venice Tips and Suggestions', it: 'Consigli e Suggerimenti di Venezia' },
    href: '/venice-tips-and-suggestions',
    is_active: true,
    sort_order: 6,
    content: null,
    map_embed: null,
    image_url: null,
    documents: [],
    downloads_enabled: false
  })
})
.then(response => response.json())
.then(data => console.log('Created:', data))
.catch(error => console.error('Error:', error))

Or run this in your terminal (if you have curl):

curl -X POST http://localhost:3005/api/admin/menu \
  -H "Content-Type: application/json" \
  -d '{
    "title": {"en": "Venice Tips and Suggestions", "it": "Consigli e Suggerimenti di Venezia"},
    "href": "/venice-tips-and-suggestions",

    "sort_order": 6,
    "content": null,
